# 功能：提醒事项

定时提醒、重复规则、Toast/对话框/系统通知、自定义提醒图；可选桌面宠物展示与到点联动。

桌面宠物实现细节见 [功能宠物.md](./功能宠物.md)。

## 功能列表

- 提醒 CRUD（标题、时间、重复、优先级）
- 主进程调度到点事件 `reminder:due`
- 渲染层 Toast 或模态对话框
- 可选 Windows 系统通知与提示音
- 延后（snooze）、忽略、清除已完成
- 顶栏铃铛入口（`reminder.enabled`）
- **桌面宠物**（`reminder.desktopPetEnabled`）：悬浮窗、精灵图导入、动画状态、随机状态、查看提醒列表、宠物侧到点弹窗

### 桌面宠物联动

```mermaid
flowchart LR
  Sched["reminder-scheduler"]
  Main["main/index.ts"]
  Hook["useReminderAlerts"]
  PetSvc["desktop-pet-service"]
  PetUI["src/pet"]

  Sched -->|due payload| Main
  Main --> Hook
  Main -->|notifyPetReminderDue| PetSvc
  PetSvc -->|pet:reminderDue| PetUI
  PetUI -->|pet:listReminders| Store
  PetUI -->|snooze/dismiss| Store
```

到点时分发逻辑（`electron/main/index.ts`）：同一 `due payload` 既发往主窗口 `reminder:due`，也在 `reminder.enabled && desktopPetEnabled` 时发往宠物窗口 `pet:reminderDue`。

右键菜单「查看提醒」→ `pet:openReminders` → 宠物页调用 `pet:listReminders` 展示未关闭提醒。
